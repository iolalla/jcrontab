JCrontab:

jcrontab is a Crontab written in JAVA.
The project objective is to provide a fully functional system to substitute Crontab in JAVA projects.

Thanks to LINUX, JAVA and Sourceforge



Provides a system to run Classes/Threads/main/... at a given time.
Provides a parser of Crontab-like files Fully compatible with UNIX-POSIX crontab files
Is an LGPL Project you can use and distribute it Freely under the LGPL License

Is a Project born in Sourceforge, and you can find all the information you need about jcrontab in Jcrontab@Sourceforge
