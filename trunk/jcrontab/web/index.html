JCrontab:

jcrontab is a Crontab written in java.
The project objective is to provide a fully functional system to substitute cron in java projects.

Thanks to linux , java and Sourceforge


Provides a system to run Classes/Threads/main/... at a given time.
Provides a parser of Crontab-like files Fully compatible with UNIX-POSIX crontab files
Stores and reads crontab table from files, DD.BB. and is designed to be extended using generic interface DataSource to store/read crontab from "any" place
Can be easily integrated with app-servers, have been tested in Tomcat, Resin, Jetty and Jboss. It has a load-on-startup servlet to make it easier
Can be used as a stand alone app to "replace" cron
Has been designed an developed to get maximum performance and extensibility.
If you want to learn more about design ideas can read Design principles
