Wanna help?

If you are interested in this project and want to help, you are welcome. The ways you can help are:

  • writing source
  • designing a better crontab
  • giving feedback
  • testing the project
  • having fun
  • and, of course,using jcrontab

If you have any suggestions, please send an email to iolalla.

If you want to subscribe to the mailing list you can do it via jcrontab-developers.
