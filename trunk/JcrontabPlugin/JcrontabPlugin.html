Jcrontab plugin

 By Israel Olala
iolalla@yahoo.com

Introduction

This plugin integrate Jcrontab into jEdit to make easier to launch tasks at a given time. Jcrontab uses Crontab POSIX format to define when a task will be executed. Can exeucte classes, call to methods (static), exectue Native Programs (tested in Linux and windows), execute beanshell macros and anything that can be called from java.

Done:

  • Launch a Crontab
  • Edit Crontab Entries from jEdit
  • Change jcrontab Properties
  • Change refresh Frequency
